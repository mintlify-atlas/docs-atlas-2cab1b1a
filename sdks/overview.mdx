## Common client properties

All SDKs expose the same set of namespaced API resources on the top-level client object:

| Property                    | Description                                      |
| :-------------------------- | :----------------------------------------------- |
| `application`               | Create and manage applications (tenants)         |
| `authentication`            | Generate dashboard access tokens                 |
| `endpoint`                  | Create, update, and delete webhook endpoints     |
| `message`                   | Send webhook messages                            |
| `messageAttempt`            | Inspect and retry delivery attempts              |
| `eventType`                 | Manage event type schemas                        |
| `integration`               | Manage third-party integrations                  |
| `operationalWebhook`        | Subscribe to Svix operational events             |
| `statistics`                | Aggregate usage statistics                       |
| `backgroundTask`            | Track long-running background jobs               |

<Note>
  All SDKs automatically detect the region from your authentication token and
  route requests to the closest regional API endpoint (US, EU, IN, CA, AU).
</Note>
