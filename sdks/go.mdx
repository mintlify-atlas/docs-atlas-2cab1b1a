## Installation

```bash
go get github.com/svix/svix-webhooks/go
```

The module path is `github.com/svix/svix-webhooks/go`. Requires Go 1.16 or later.

## Initializing the client

```go
import svix "github.com/svix/svix-webhooks/go"

sv, err := svix.New("AUTH_TOKEN", nil)
if err != nil {
    log.Fatal(err)
}
```

The client automatically detects the region suffix on your token (`.us`, `.eu`, `.in`, `.ca`, `.au`) and routes requests to the correct regional API endpoint.

### `SvixOptions`

```go
import (
    "net/url"
    "time"

    svix "github.com/svix/svix-webhooks/go"
)

serverURL, _ := url.Parse("https://api.eu.svix.com")
retrySchedule := []time.Duration{
    50 * time.Millisecond,
    100 * time.Millisecond,
    200 * time.Millisecond,
}

sv, err := svix.New("AUTH_TOKEN", &svix.SvixOptions{
    ServerUrl:     serverURL,
    RetrySchedule: &retrySchedule, // up to 5 entries
    Debug:         false,
})
```

| Field               | Type                          | Description                                                  |
| :------------------ | :---------------------------- | :----------------------------------------------------------- |
| `ServerUrl`         | `*url.URL`                    | Override the API base URL                                    |
| `HTTPClient`        | `*http.Client`                | Custom HTTP client                                           |
| `TransportWrapper`  | `func(http.RoundTripper) http.RoundTripper` | Wrap the default transport                   |
| `RetrySchedule`     | `*[]time.Duration`            | Custom retry delay schedule (max 5 entries)                  |
| `Debug`             | `bool`                        | Enable debug logging                                         |

## Sending a webhook

<Steps>

### Create an application

```go
import "github.com/svix/svix-webhooks/go/models"

app, err := sv.Application.Create(ctx, models.ApplicationIn{
    Name: "my-app",
}, nil)
if err != nil {
    log.Fatal(err)
}
fmt.Println(app.Id) // app_...
```

### Send a message

```go
msg, err := sv.Message.Create(ctx, app.Id, models.MessageIn{
    EventType: "invoice.paid",
    Payload: map[string]any{
        "type":      "invoice.paid",
        "invoiceId": "invoice_123",
        "amount":    9900,
    },
}, nil)
if err != nil {
    log.Fatal(err)
}
fmt.Println(msg.Id) // msg_...
```

</Steps>

## Verifying a webhook

```go
import (
    "net/http"

    svix "github.com/svix/svix-webhooks/go"
)

func webhookHandler(w http.ResponseWriter, r *http.Request) {
    body, err := io.ReadAll(r.Body)
    if err != nil {
        http.Error(w, "bad request", http.StatusBadRequest)
        return
    }

    wh, err := svix.NewWebhook("whsec_<your-endpoint-secret>")
    if err != nil {
        http.Error(w, "internal error", http.StatusInternalServerError)
        return
    }

    if err = wh.Verify(body, r.Header); err != nil {
        http.Error(w, "unauthorized", http.StatusUnauthorized)
        return
    }

    // payload is verified — handle it
    w.WriteHeader(http.StatusNoContent)
}
```

<Warning>
  Read the raw body **before** any JSON parsing. Pass `r.Header` directly; the
  verifier reads `svix-id`, `svix-timestamp`, and `svix-signature` from it.
</Warning>

<Note>
  `NewWebhook` accepts secrets with or without the `whsec_` prefix.
  Use `NewWebhookRaw` to supply a raw byte slice instead.
</Note>
